# SHOPPING WEBSITE

 **Live Demo:** [Click here](https://rajd-proj-shop-1.vercel.app/)

##  Description
A web application built to mimic a product and shopping. This project highlights my ability to build a good UI and responsive layout using modern React practices.

##  Features
* **Core Functionality:** To fetch some product details from the API using API calls
* **Responsive Design:** Fully optimized for mobile and desktop views.
* **State Management:** Efficiently handled using React Hooks.
---

##  Technical Setup
  ### Installation
    1. Clone the repository
    2. Run `npm install`
    3. Start the project with `npm start`
## Dependencies 
  * **FRONTEND:** React.js
  * **DEPLOYMENT:** Vercel
